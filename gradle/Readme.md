Notes
=======

Gradle version 6.3

Instalation (windows)
=====================

* Download binary
* extract
* add bin folder to PATH environment variable.
* check installation: gradle --v


Visual Studio Code
===================

## Plugins

* Java Language Support.
* Java Code Generators
* Gradle Language Support.
* Gradle Task


Commands
==========

* create project

```
gradle init
```

* compile

```
gradle build
```

* clean

```
gradle clean
```

* final jar in directory: `build\libs\`

* run test (and display info)

```
gradle test -i
```
